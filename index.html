PowerShell Scripts

Active Directory & Exchange

Back in 2016-17, while doing Windows support in a corporate ennvironment using Active Directory (AD) & Exchange mail server, I discovered that Powershell could be used to create scripts for reptitive tasks. So I taught myself enough Powershell to write scripts for creating public folders and shared mailboxes. About a year later I rewrote (refactored) the shared mailbox script, as I had improved my knowledge & understanding of Powershell

Then the company we were supporting had a bit of a restructure and my manager had to move a lot of user objects into differnt departments units under different managers. This also meant that he had to transfer ownership of security groups from one manager to another. He was doing all this extra admin work in his evenings, so I created a few scripts to help him do this more quickly & easily.

Resources

Here are a few resources that I have discovered and used to learn about web development -

  • Codecademy - courses in HTML, CSS, JavaScript, PHP etc
  • Internetting is Hard - tutorials about HTML & CSS
  • FreeCodeCamp - especially the 'Responsive Web Design Certification' section with its Project sub-sections
  • SQL Zoo Online tutorial about learning SQL
  • DB Fiddle An online Database sandbox
  • Free webinars about web dev topics via Eventbrite - just search for a topic and use the Filters for Price & Date eg Free Git events this week. FYI its an international site so you may need to convert between time zones
  • Free introductory workshop webinars via Le Wagon
  • Intro to GitHub by Dr R Ainsworth. GitHub is cloud hosting for files/projects that are version controlled and can be worked on by multiple people
  • Intro to Git Version Control by A Skoulikari. Git is version control software that runs locally on your computer
